🔄 Meta Refresh 重定向示例

📝 知识点说明:

Meta Refresh 重定向是通过 HTML 的 <meta> 标签实现的客户端重定向

语法: <meta http-equiv="Refresh" content="秒数; URL=目标地址">

特点:

  • 客户端重定向，在浏览器端执行
  • 优先级低于 HTTP 状态码重定向
  • 支持延时跳转，用户体验较好
  • SEO 友好性一般

页面将在 5 秒后自动跳转到目标页面

🔗 优先级对比:

1. HTTP 状态码重定向 (最高优先级)

2. Meta Refresh 重定向 (中等优先级)

3. JavaScript 重定向 (最低优先级)
