🎯 重定向优先级测试

这个页面同时包含了三种重定向方式，用于测试它们的优先级：

🚨 1. HTTP 状态码重定向 (最高优先级)

服务器端重定向，通过 HTTP 响应状态码实现

HTTP/1.1 302 Found
Location: /target

⏰ 2. Meta Refresh 重定向 (中等优先级)

HTML Meta 标签重定向，当前页面设置了 10 秒后跳转

<meta http-equiv="Refresh" content="10; URL=/target?from=meta-refresh">

Meta Refresh 倒计时: 10 秒

💻 3. JavaScript 重定向 (最低优先级)

客户端脚本重定向，当前页面设置了 15 秒后跳转

setTimeout(() => {
  window.location.href = '/target?from=javascript';
}, 15000);

JavaScript 倒计时: 15 秒
