## dns 查询
```bash
# 查询某个 dns 服务器，关于某个域名的信息，用来判断该 dns 是否可以正常解析某域名
# 例如 查询 8.8.8.8 dns 服务器对 baidu.com 这个域名的返回
dig @8.8.8.8 baidu.com

```
